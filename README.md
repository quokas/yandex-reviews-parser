# Сервис мониторинга и кэширования отзывов с Яндекс.Карт

Fullstack-приложение для автоматического сбора, фильтрации и отображения отзывов организаций. Проект состоит из бэкенда на Laravel, фронтенда на Vue 3 и независимого парсера на Puppeteer.

## 🛠 Технологический стек
* **Backend:** Laravel 11 (API, Eloquent ORM)
* **Frontend:** Vue 3 (Composition API), Axios, Tailwind CSS, Vite
* **Database:** SQLite (для локального кэширования данных)
* **Automation:** Node.js, Puppeteer (скрипт для веб-скрейпинга)

## ⚙️ Как это устроено (Архитектура)

1. **Сбор данных (Puppeteer):** Скрипт `parcer.cjs` открывает страницу организации, имитирует поведение реального пользователя (плавный скролл, случайные задержки) для обхода блокировок и собирает чистый текст отзывов, отсекая лишние элементы интерфейса Яндекса.
2. **Обработка и дедупликация (Laravel):** Полученные данные отправляются на бэкенд. Перед сохранением Laravel очищает тексты от обрезков (вроде строк `... еще`), нормализует их и проверяет на дубликаты по хэш-ключам. Это предотвращает повторную запись одних и тех же отзывов.
3. **Оптимизация базы данных:** Сохранение данных в SQLite происходит чанками (порциями по 100 записей), чтобы снизить нагрузку на оперативную память.
4. **Интерфейс (Vue 3):** Адаптивное SPA-приложение, которое отображает сохраненные отзывы из базы данных с поддержкой пагинации.

## 🔒 Безопасность и сессии
Вся пользовательская дата, куки и временные файлы браузера сохраняются локально в папке `yandex_user_data` и автоматически игнорируются Git. Ваши личные данные авторизации защищены и никогда не попадут в публичный репозиторий.

## 🚀 Инструкция по локальному развертыванию

### 1. Клонирование проекта
```bash
git clone https://github.com
cd yandex-reviews-parser
```

### 2. Настройка бэкенда (Laravel)
```bash
composer install
cp .env.example .env
php artisan key:generate
```
*Примечание: Убедитесь, что в файле `.env` в качестве драйвера БД указан `DB_CONNECTION=sqlite`.*

### 3. Настройка базы данных и сидов
```bash
touch database/database.sqlite
php artisan migrate:fresh --seed
```

### 4. Настройка фронтенда и парсера (Node.js)
```bash
npm install
npm run build
```

### 5. Запуск проекта
```bash
php artisan serve
```
После этого приложение будет доступно по адресу: `http://127.0.0.1:8000`
